## Run

运行命令启动前端 如：

```shell
yarn f
# or
npm run f
```

所有命令及作用：

```shell
yarn i-all: "安装前端和 Hardhat 的依赖",
yarn f: "启动前端",
yarn hh: "启动 Hardhat 网络",
yarn hh-compile: "编译合约",
yarn hh-dev-l: "运行 部署脚本",
yarn hh-clear: "清理 编译的合约文件 和 ABI文件",
yarn hh-test: "运行 测试脚本"
yarn hh-dev: "运行 清理编译/ABI文件 -> 部署脚本",
```
